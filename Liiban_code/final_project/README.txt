Below are the required steps for running the Web application
**********************************************************************************************************************

If you have Anaconda and Python installed on your computer
1) Open CMD or Terminal
2) Conda create -n environment_name python=3.7	 #This will create a virtual environment with the python used
3) activate environment_name   		         #This will activate your virtual environment that you just created
4) pip install -r requirements.txt      # this will install the required packages to run the application
5) python manage.py runserver 				# this will rune the local server for the project in order to 
6) open Google chrome 					# open the google chrome web browser
7) http://127.0.0.1:8000/home      # enter this into your browser URL, this can also be found in the terminal after					     server has been run
